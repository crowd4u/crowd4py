Crowd4py: Crowd4u for Python!
============================
API crient for Crowd4u (http://crowd4u.org/ja/).

## Installation

```
git clone https://github.com/crowd4u/crowd4py.git
cd crowd4py
python setup.py install
python setup.py test
```

## Usage
```
cd sample/
# create config.ini
python Phase1-algorithm-sample.py
```
